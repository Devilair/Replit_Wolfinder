Dashboard Amministrativa Wolfinder: Specifiche Tecniche Complete
1. Dashboard Principale (Home)
Pannello Superiore: KPI Critici
•	Metriche in tempo reale con confronto periodo precedente: 
o	Utenti attivi (oggi, settimana, mese)
o	Nuove recensioni (verificate vs non verificate)
o	Nuovi professionisti registrati
o	Tasso di conversione abbonamenti
o	Ricavi correnti (MTD) con proiezione mensile
Grafici Principali
•	Grafico principale interattivo con toggle per visualizzare: 
o	Recensioni totali/verificate/rifiutate (trend temporale)
o	Registrazioni professionisti per categoria
o	Conversione da free a piani a pagamento
o	Engagement utenti (visite, ricerche, interazioni)
Attività Critiche
•	Code di lavoro con contatori: 
o	Recensioni in attesa di verifica (prioritizzate per rating)
o	Richieste di verifica professionisti
o	Segnalazioni contenuti inappropriati
o	Richieste di supporto non gestite
Avvisi e Notifiche
•	Sistema di alerting con livelli di priorità: 
o	Attività sospette (pattern di recensioni anomale)
o	Picchi di traffico o utilizzo insoliti
o	Problemi di performance del sistema
o	Threshold raggiunti (es. >50 recensioni in attesa)
2. Gestione Professionisti
Lista Professionisti Master
•	Tabella avanzata con: 
o	ID, nome, professione, località, data registrazione
o	Status verifica (verificato, in attesa, respinto)
o	Rating medio e numero recensioni
o	Piano abbonamento attivo
o	Indicatori di attività (ultimo accesso, % completamento profilo)
o	Flag per professionisti problematici
Filtri e Ricerca
•	Sistema di filtri avanzati: 
o	Multi-select per categorie professionali
o	Filtro geografico (regione, provincia, città)
o	Filtro per status verifica e completezza profilo
o	Filtro per piano abbonamento
o	Ricerca full-text con suggerimenti
o	Filtri salvabili per uso frequente
Scheda Dettaglio Professionista
•	Panoramica completa: 
o	Dati anagrafici e di contatto completi
o	Storico verifiche con documenti caricati
o	Timeline attività sulla piattaforma
o	Tutte le recensioni ricevute con filtri
o	Statistiche di performance (visualizzazioni profilo, CTR, confronto con media categoria)
o	Log comunicazioni e note amministrative
Azioni Amministrative
•	Suite completa di strumenti: 
o	Verifica/rifiuto credenziali con feedback
o	Modifica diretta informazioni profilo
o	Sospensione temporanea account (con timer)
o	Ban permanente con motivazione
o	Gestione manuale abbonamento
o	Invio comunicazioni dirette
o	Generazione report personalizzati
3. Gestione Recensioni
Dashboard Recensioni
•	Panoramica status recensioni:

o	Tempo medio di verifica

•	Sistema di prioritizzazione intelligente:

o	Indicatori tempo in coda

o	Accesso a storico utente e professionista
o	Quick-actions per approvazione/rifiuto
o	Template risposta per rifiuti comuni
o	Sistema di flag per revisione senior
o	OCR per estrazione automatica dati da documenti
Gestione Segnalazioni
•	Sistema completo: 
o	Vista aggregata segnalazioni per recensione

o	Sistema di notifica agli interessati
